# data-engineering-project

## 📌 Overview
This project builds a data pipeline using Airflow, Spark, HDFS, and Snowflake.

## 🛠 Tech Stack
- Apache Airflow
- Apache Spark
- HDFS
- YARN
- Snowflake

## 🔄 Pipeline
1. Load data from CSV
2. Store in HDFS
3. Process using Spark
4. Load into Snowflake
